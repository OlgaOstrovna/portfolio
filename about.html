Here I would like to provide you with the main details of my project workflow.

Time spent: 30 hours.

Tools used for the creation of this website: NinjaMock, Sublime, Command Prompt, Browser-sync, Google Browser.

Difficulties: Since I am a newcomer to the field of programming it took me some time and effort to understand the programming approach and to look for solutions for all the challenges.

My first step was thinking and developing an idea for my future website. At this stage, this is crucial to visualize the idea as a layout. For doing this I used a nice resource NinjaMock.

I decided to create 3 pages:

1. “About” - brief description of the website and it’s idea.

2. “Projects” - some information about each of my projects with photos and links.

3. “About the Portfolio” - contains information about the framework and all the main steps of the website creation.

Note: since the main task of this assignment was the creation of a simple website, this portfolio website can be advanced and developed as well as the information it contains.

I will introduce my further work in main steps providing a brief description together with screenshots.

Connecting project folder with browser server using Command Prompt in order to conduct the updates on the website after each editing in Sublime automatically.

Starting coding the main structure of the website with Sublime.

Using Lorem Ipsum as a generator of typographic layout in order to plan and organize the page space.

Embedding a picture to the Home page as well as embedding the background picture. The background picture is static, means it does not move during scrolling down through the page.

Creation of a CSS file in order to set up the design like fonts, colours etc.

Setup of a centre position of the picture and text in order to have a proportional allocation of each element.

Setup of the function for navigation (navigation panel/toolbar) - “header”.

Setup of the work and access to social media links (Twitter, GitHub, LinkedIn) - “footer”.

Connection between all the pages for an acces to each page with navigtion panel.

Creation of a project.html file in order to store there all the information about my projects (“Projects” page).

Creation of cards - simple div elements (images+information)

Since I wanted modal windows appear by clicking on each card I used javascript code that was suggested on w3school.

Thus, this task was both challenging and very interesting for me at the same time. It was a good dive into programming for me. Definitely, I have faced some problems and questions, but the process of googling helped me find the solutions.

So now you are very welcome to my portfolio website!

Enjoy! :)
